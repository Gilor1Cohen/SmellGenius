It delivers personalized perfume suggestions based on user preferences, purchase history, and contextual scenarios (e.g. date night, office, summer). ## üöÄ Getting Started

### Clone the repo

```bash
git clone https://github.com/Gilor1Cohen/SmellGenius
cd SmellGenius
```

### Install dependencies

#### Server

```bash
cd server
npm install
```

#### Client

```bash
cd client
npm install
```

### Run the app

```bash
# Terminal 1 - server
cd server
nodemon index

# Terminal 2 - client
cd client
npm run dev
```
